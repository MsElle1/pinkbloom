# PinkBloom Flower Shop

## Project Description
PinkBloom is a simple web application built with Spring Boot that allows users to create an account, log in, and view a flower shop interface. The system uses PostgreSQL for user authentication and is deployed using Railway.

## Technologies Used
- Java
- Spring Boot
- Thymeleaf
- PostgreSQL
- Railway (deployment)
- GitHub (version control)

## Features
- User signup
- User login
- Flower shop interface
- Logout functionality
- PostgreSQL database storage

## Database Structure
Table: users

| Field | Type |
|------|------|
| id | Long |
| email | String |
| password | String |

## How to Run Locally

1. Clone the repository

2. Configure PostgreSQL
  
3. Run the application
4. Open browser
